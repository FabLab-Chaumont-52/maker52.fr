Pour avoir accès au forums de travail, merci de suivre les instructions suivantes :

  1. Rendez-vous sur la page Framavox
  2. Cliquez sur le bouton "Login / Create account"
  1. Cliquez sur le champ "Email"
  1. Renseignez votre email.
  2. Cliquez sur le bouton "Continue with Email"
  1. Renseignez votre Nom d'utilisateur.
  2. Cliquez sur le bouton "CREATE ACCOUNT"

Un mail vous est envoyé.

  1. Relevez votre mail et cliquez sur le bouton "Sign in as Votre Nom Utilisateur"

Vous étes de renvoyés sur le site :

  1. Cliquez sur le bouton "Sign in as Votre Nom Utilisateur"
  1. Cliquez sur le bouton "Login / Create account"

Oufff!!!

Si vous faites déjà partie d'un ou plusieurs groupes, vous arrivez sur cette page.

La liste des groupes dont vous faites partie se trouve dans la colonne de gauche.

Si vous avez connaissance d'un groupe FabLab et que vous ne le voyez pas dans la colonne de gauche, vous pouvez envoyer un mail à {{ site.email }} en précisant bien l'adresse email avec laquelle vous vous identifiez sur Framavox et le groupe dont vous souhaitez faire partie.

ou

Si vous ne faites partie d'aucun groupe, vous arrivez sur cette page (aucun groupe Maker 52 ne figure dans la colonne de gauche).

Vous pouvez alors envoyer un gentil mail à {{ site.email }} en confirmant bien l'adresse email avec laquelle vous vous identifiez sur Framavox.
